Welcome to the Code Formatting App!

  • To start using the app simply paste the code you wish to format into the text box and select the language the code is written in.
  • Then press format and your formated code will appear at the end of the screen.
  • If you would just like your code coloured and not formatted just tick the box that says 'just colouring'.

    More Options

  • If you would like to further customise your code just click the more options link where you can choose from even more programming languages and select the indentaion you would prefer.
