Welcome to the Code Formatting App!

  • To start using the app simply paste the code you wish to format into the text box and select the language the code is written in.
  • You can select/unselect the tickboxes for "Formatting", "Colouring" or "Comments", based on whether you want your code to be formatted, highlighted based on keywords, or to contain any comments
  • Then press format and your formatted code will appear at the end of the screen.

    More Options

  • If you would like to further customise your code just click the more options link where you can choose from even more programming languages and select the indentation you would prefer.
